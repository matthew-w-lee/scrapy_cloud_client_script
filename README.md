# Scrapy Cloud Client Script
## About
Simple Python Flask application used to send web scraping commencement requests to Scrapy's cloud service API. It automates the retreival of status information and the saving of scrape results from several crawlers deployed on the cloud service.

This project is no longer maintained and may or may not be operational.

## Built With
* [Scrapy](https://github.com/scrapy/scrapy)
